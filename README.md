# sajida bagha G3  – Backend (PHP)

Cette partie concerne la gestion de l’inscription des utilisateurs
et la connexion à la base de données.

## Fichiers réalisés
- register.php : traitement du formulaire d’inscription
- config.php : configuration et connexion à la base de données

## Technologies utilisées
- PHP
- MySQL

## Fonctionnalités
- Enregistrement des utilisateurs
- Connexion sécurisée à la base de données
- Gestion des erreurs de connexion
  
## Rôle
Responsable de l’inscription et de la configuration backend.
